# bc547-blinker

**Ein analoger DIY-Blinker mit zwei Transistoren – ganz ohne Code.**

## Bauteile
- Transistor: BC547 (2x)
- Kondensator: 470 µF (2x)
- Widerstand (Basis): 10 kΩ (2x)
- Widerstand (LED): 220 Ω (2x)
- LED: rot & blau, 5mm

## Kurzbeschreibung
Klassischer astabiler Multivibrator im symmetrischen Aufbau. Die zwei BC547 schalten sich per Rückkopplung gegenseitig und lassen die LEDs abwechselnd blinken. Das Timing übernehmen die Kondensatoren zusammen mit den Basis-Widerständen.
